After Task 1, the hook signature is:

```typescript
export type RealtimePhase = 'idle' | 'connecting' | 'listening' | 'responding' | 'error';

export interface RealtimeStatus {
  phase: RealtimePhase;
  transcript: string;
  responseText: string;
  error: string | null;
}

export function useRealtimeVoice(): {
  status: RealtimeStatus;
  analyserRef: React.RefObject<AnalyserNode | null>;
  connect: () => Promise<void>;
  disconnect: () => void;
  isConnected: boolean;
};
```

Connect → connecting → listening (after `session.ready`) → responding (during TTS) → listening. `connect()` MUST be invoked from a user gesture (click handler) because it calls `getUserMedia` synchronously before any setState (mobile gesture-window requirement).

ws-server endpoint contract (from ws-server/src/index.ts):
- Listens on `process.env.PORT || 8080`
- Upgrades WebSockets at path `/ws` only
- Health check at `GET /health` returns "ok"
- Client sends `{type: 'session.start'}` first, then streams `{type: 'audio.append', data: <base64 PCM16>}`
- Server sends `session.ready`, `transcript.partial`, `transcript.final`, `response.text.delta`, `response.audio.delta`, `response.done`, `error`
</interfaces>

import { useRealtimeVoice } from '@/hooks/useRealtimeVoice';

export default function Home() {
  const { status, connect, disconnect, isConnected } = useRealtimeVoice();

  const handleClick = () => {
    if (isConnected || status.phase === 'connecting') {
      disconnect();
    } else {
      // MUST be a direct call inside the click handler — getUserMedia requires
      // the user-gesture activation window (especially on mobile Chrome).
      void connect();
    }
  };

  const buttonLabel = isConnected || status.phase === 'connecting' ? 'Disconnect' : 'Connect';

  return (
    <main className="flex min-h-screen flex-col items-center justify-center gap-6 bg-black p-8 font-mono text-zinc-100">
      <h1 className="text-2xl tracking-tight">JTAC Trainer — Voice Smoke Test</h1>

      <button
        type="button"
        onClick={handleClick}
        className="rounded border border-zinc-600 bg-zinc-900 px-6 py-3 text-base hover:bg-zinc-800 active:bg-zinc-700"
      >
        {buttonLabel}
      </button>

      <div data-testid="phase" className="text-sm text-zinc-400">
        phase: {status.phase}
      </div>

      {status.error && (
        <div data-testid="error" className="max-w-xl text-sm text-red-400">
          error: {status.error}
        </div>
      )}

      <section className="flex w-full max-w-xl flex-col gap-2">
